# Release Notes v1.0.0

Data de lançamento: 02/05/2026

## Novidades

### Painel lateral (Sidebar)

- Novo painel **Comparar Branches** na barra de atividade do VS Code
- Campo **Branch 1** e **Branch 2** para seleção de branches via QuickPick
- Listagem dos últimos 20 commits de cada branch selecionado
- Seleção de commit específico por branch (slot) para comparação pontual
- Botão **Comparar** (▶) e **Atualizar** (↻) na barra do painel

### Painel de diff (Webview)

- Comparação lado a lado dos arquivos alterados entre dois refs
- Destaque de linhas adicionadas (`+`) e removidas (`−`)
- **Destaque inline** no nível de caractere para linhas modificadas (semelhante ao diff nativo do VS Code)
- Sincronização de scroll vertical e horizontal entre as colunas esquerda e direita
- Modo **fullscreen** por arquivo — botão de expandir em cada bloco de arquivo; tecla `ESC` para sair
- Selects no topo do painel sincronizados com a seleção feita na sidebar ao comparar

### Comandos registrados

| Comando                                          | Descrição                               | Paleta |
| ------------------------------------------------ | --------------------------------------- | ------ |
| `Git Compare: Comparar Commits`                  | Abre o painel de comparação             | Sim    |
| `Git Compare: Selecionar Branch`                 | Seleciona branch para um slot           | Não    |
| `Git Compare: Selecionar Commit para Comparação` | Seleciona commit para um slot           | Não    |
| `Git Compare: Comparar Seleção da Lateral`       | Compara os refs selecionados na sidebar | Não    |
| `Git Compare: Atualizar Sidebar Git Compare`     | Recarrega a árvore lateral              | Não    |

### Serviço Git (interno)

- Detecção automática de repositório Git no workspace
- Listagem de branches locais e remotos
- Listagem de commits por ref com formato `hash \x1f autor \x1f data \x1f mensagem`
- Listagem de arquivos alterados entre dois refs
- Leitura de conteúdo de arquivo em um ref específico
- Diff completo e diff por arquivo entre dois refs
- Estatísticas de diff (linhas adicionadas/removidas por arquivo)

### Requisitos mínimos

- Visual Studio Code `^1.95.0`
- Repositório Git no workspace aberto
