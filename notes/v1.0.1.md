# Release Notes v1.0.1

Data de lançamento: 03/05/2026

## Correções

### Correções de bugs

- Corrigido erro no CI (Trivy): falha `permission denied` ao escrever relatórios em `/tmp/ci-reports` quando o container roda com UID não-root. Atualização no workflow adiciona `chmod 0777` em `${{ runner.temp }}/ci-reports` antes do mount, evitando a falha do scanner.
- Corrigido problema de exibição dos GIFs no Marketplace: imagens no README não eram carregadas por usarem caminhos relativos; o README agora referencia as imagens por URLs absolutos em `raw.githubusercontent.com`.
- Corrigido empacotamento do VSIX: GIFs estavam sendo incluídos no pacote e inflavam o `.vsix`; `.vscodeignore` foi atualizado para excluir `media/*.gif` do empacotamento.
- Corrigido comportamento de erro no `GitService`: remoção de `vscode.window.showErrorMessage` para evitar mensagens duplicadas ao usuário e permitir tratamento centralizado de exceções.
- Ajuste no workflow de publicação: o pipeline agora lê a versão a partir de `package.json`, detecta se a tag `vX.Y.Z` já existe e impede publicação duplicada; quando for versão nova, cria tag e GitHub Release automaticamente (com conteúdo de `notes/vX.Y.Z.md` quando presente).

### Outros ajustes

- Atualizações na documentação e demonstrações visuais no `README.md` (caminhos das imagens e textos explicativos).
- Atualização de metadados e incremento de versão para `1.0.1` em `package.json`.

## Pendências / próximas versões

- Incluir `media/icon.png` (128×128 px) para atender requisitos do Marketplace.
- Rever e confirmar screenshots publicadas no Marketplace (caso o Marketplace limite GIFs grandes).
